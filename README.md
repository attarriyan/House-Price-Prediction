# House Price Prediction

## Project Overview
This project predicts house prices using Linear Regression.

## Dataset
Kaggle House Prices Dataset

## Features Used
- GrLivArea
- BedroomAbvGr
- FullBath
- LotArea
- OverallQual
- YearBuilt

## Libraries
- Pandas
- Scikit-learn

## Results
- MAE: 26674
- R² Score: 0.779
